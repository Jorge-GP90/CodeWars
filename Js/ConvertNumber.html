JavaScript Convert reversed number into a array

Methods

  • Array.from()
  • num.toString()
  • num.reverse()
  • num.map()
  • num.split('')

Test Number: 123


First way: Array.from(String(num), Number).reverse();


Second way: String(num).split('').map(Number).reverse();


Third way: (num + '').split('').map(Number).reverse();
